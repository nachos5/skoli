    Keys:

  • A and D for left and right movement.
  • Spacebar for SUPERBOOST, careful though, it will reduce your paddle width!

      • Utils:

    • P for pause.
    • O for single-step.
    • Q for quitting.

    Cool stuff:

  • Superboost by clicking spacebar, paddle shrinks while doing so.
  • Invulnerability after ball spawn, indicated by the ball flickering, which lasts
    until the ball collides with the floor or the paddle.
  • Sound. No pesky audiofiles. Using web audio API to generate simple oscillators. Added some randomness (only to the bricks though) to make it a bit more dynamic.
  • Score system, keeps track of the all time high score (with local storage),
    current session high score and finally the current stage.
  • Lifes. There are 3 lifes each stage, indicated by the heart sprites.
  • Moving and shrinking the canvas.
  • Difficulty. Each stage you progress, the game gets harder by increasing the speed. Of everything.
    Also the ball gets gradually faster the longer it has been in play (since reset).
  • Bricks constantly changing to a new random color. Very cool.

    Not so cool:

  • 'C' (clear-toggle) can't work in my game as the canvas clears itself when it resizes.
  • Collisions. Tried working on them and they are decent, but definitely buggy sometimes.
